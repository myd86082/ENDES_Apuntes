## :point_down: Lenguaje de etiquetado Markdown :point_down:

**Markdown es un lenguaje de marcado al estilo de HTML** (pero mucho más sencillo), creado por John Gruber en 2004, con el objetivo de facilitar la redacción de documentos estructurados sin necesidad de escribir código HTML.

Este formato se usa ampliamente no solo en GitHub, sino también en:
 - Documentación técnica (manuales, wikis, proyectos).
 - Blogs o gestores de contenido.
 - Entornos educativos o de aprendizaje online.
